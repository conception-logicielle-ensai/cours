Ce TP n'a pas intention à être fini, il est plus une proposition de travail dans le temps associé

Ce TP regroupe des notions d'architecture applicative, webservice API et frontend.

## Architecture

- Pour les équipes de 4 et les équipes motivées, avoir une architecture type Backend / Frontend

Donc: Un module React / Un module API (django ou fastapi)

- Pour les autres : avoir une architecture API ou rendu côté serveur


## Objectif de la séance 

- Mettre en place les modules, un README pour les lancer

**Sans penser fonctionnel, on vous propose de mettre en place l'architecture associée a votre projet**

- Pour les personnes faisant un frontend, faire un appel API vers le backend
- Pour les API avoir un endpoint
- Pour les rendus côté serveur avoir une page


## Objectif Bonus

Mettre en place les contrôles automatiques en local (githooks) et en distant (actions) associées au frontend.
